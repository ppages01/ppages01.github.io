ETL Big Data - A 2 phase project with an extra 3rd Machine Learning component.. This involved reading and loading an Amazon review’s data file, cleaning up the columns and transforming it so it can be loaded into a Postgress database. Phase 2 involved analyzing and summarizing  the data while using visualization tools. Phase 3 implemented a Naive Bayes learning machine model, analyzing the review text and forming a model prediction on whether a review was "Good" or" Bad".
